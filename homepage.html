To Do Application

There is a range of functions that you can use:

Create a to do (createToDo("string","string").
Remove an item from the to do list (removeToDo(i)) this can remove a string or by index.
View all todo's (showAll()).
Mark an item in the todo as completed(markAsCompleted()) can take a string or by index.
Remove all items from to do list (removeAllToDos()).
Show all uncompleted todo's (showUncomplete()).
Show all Completed todo's (showComplete()).
